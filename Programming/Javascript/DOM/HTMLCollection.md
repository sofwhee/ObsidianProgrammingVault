Is a live, ordered collection of the DOM elements which are children of `node`. You can access the individual child nodes in the collection by using either the [`item()`](https://developer.mozilla.org/en-US/docs/Web/API/HTMLCollection/item "item()") method on the collection, or by using JavaScript array-style notation.
