## Concept

The fast alternative to React.

Its focus is the bundle size. Stricted bundle size rules of any framework.
Delete as much code as you add to reduce bundle size.
It's not React but you can copy paste most from one to the other.

HTM support:

You can use a template literal in JS to call function.

By doing this you can write effectively JSX inside of JS, not need a transpilation step, and this gets parsed at runtime.

You can write a project that uses Preact and looks the same as you would expect React to work, *wihout needing a build step at all.*
