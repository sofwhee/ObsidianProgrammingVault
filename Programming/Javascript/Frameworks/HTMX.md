## Concept

Allows for non JS devs to make interactive web apps.

If you wanna have a button with loading state when you click on it
or twitter clone where things load without refreshing page...

HTMX wrote that JS for you and now you can just bind to a template directory.

You can write for it in any language that isn't JS. Even Python and Ruby.
