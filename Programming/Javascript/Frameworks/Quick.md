Slightly better than vanilla JS
## Concept

Modern JS frameworks have problems
- loading way too much JS
- time to interact getting slowly
- data waterfalls pop in asynchronously instead of all at once

Created to tackle these problems.

**Hydration:** Normal JS frameworks thinking.
- Given some html 
- and then given some JS, 
- the JS will "catch up" to the HTML by re-rerendering all of the parts.

**Resumability**: Skip a lot of those steps
