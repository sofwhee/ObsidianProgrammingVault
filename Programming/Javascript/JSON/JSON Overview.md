JavaScript Object Notation (JSON)

Used for sending data between web applications.

Usage:
- Sent as string
- You convert the string to a object
- Now you can access the data as a JS object

Facts:
- Text file `.json`
- Purely a string with a specified data format
- Only properties, no methods
- Single quotes are not valid inside JSON
- https://jsonlint.com/ to check for misplaced commas/colons
- Arrays / objects are not the only valid data types
- Only quoted strings can be used as properties for JSON
