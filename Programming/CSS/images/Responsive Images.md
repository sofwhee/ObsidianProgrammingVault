```
background-image: url("cat-front.png");
background-image: url("assets/cat-front.png");
```

background-image (meant for background-image)
- background-position:
	- "center"
- background-size:
	- "cover" - resizes the image to fill container
- background-repeat:
	- no-repeat

object-fit (meant for img tags)
- specify h + w
- then tell how to fit itself to those dimensions
- "fill" by default, stretches to fit
