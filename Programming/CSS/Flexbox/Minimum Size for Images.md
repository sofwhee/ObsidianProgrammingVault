#CSS

image {
	max-height: 95vh (how big it can get)
	width: auto; (this keeps the aspect ratio)
}

flex container {
	min-width: fit-content (make it fit the content of the container)
}


Intrinsic-size: The raw size of an element without any CSS.
Max-content: The maximum [intrinsic size](https://developer.mozilla.org/en-US/docs/Glossary/Intrinsic_Size) of the element's content.
Fit-content: The box will use the available space, but never more than [`max-content`](https://developer.mozilla.org/en-US/docs/Web/CSS/max-content).
