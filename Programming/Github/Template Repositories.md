Making a GitHub template repository!

1. Under where you name a repository, there is a checkbox to make it a template.
2. Now when you make a new repo, the Repository Template dropdown will list any of your templates to select. 
3. Selecting one means your new repo will be a copy of the chosen template.

When you find yourself reusing a lot of setup code for multiple projects, you can make a new repository with all of the setup code you need and mark it as a template,

Create one:

`gh repo create -p <repository>`
