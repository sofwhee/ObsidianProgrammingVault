## 3 Accepted `type`s

`submit` (default)

Will always try to make a new request and submit data back to the server. If you're not submitting data, the `type` attribute should always be specified otherwise to avoid unwanted effects.

```html
<button type="submit">Submit</button>
```

`reset`

Clears all the data the user has entered into the form and sets all the inputs in the form back to what they were initially.

```html
<button type="reset">Reset</button>
```

Generic AKA `button`

Commonly used with JS for creating interactive UI’s.

```html
<button type="button">Click to Toggle</button>
```
