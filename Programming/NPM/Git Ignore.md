It's customary to add `node_modules` to `.gitignore` since it can get really big. `dist` is often ignored.

When creating a new repo on GitHub, there is an option to specify a `.gitignore` template. 

There are many templates out there that include common files and directories that are not typically tracked based on the type of project or language used. 

For JavaScript projects, there is a `node`template that includes `node_modules` and `dist`.
