Contains all information about `dependencies`. (files needed for app to run)

You can add a `package.json` file to your package to make it easy for others to manage and install.

A `package.json` file:
- lists the packages your project depends on
- specifies versions of a package that your project can use using [semantic versioning rules](https://docs.npmjs.com/about-semantic-versioning)
- makes your build reproducible, and therefore easier to share with other developers
