Hard edges capture the eye first. So they tend to be in focal points
Soft edges support that decision by being the opposite.

Hard edges can contain a lot of information about a form.
That means that the form can dissolve into soft edges and still be understood.
This is called lost and found.

A painter can use some hard edges to communicate and get people going, and then lose it for fun, back and forth.

Soft edges allow the eye to group a bunch of stuff, file it away, and then focus our attention on the focal hard edged figure.

## painting your own edges

You can just spam color picker between two values and blend them in with the same brush.
with airbrush you just increase/decrease brush size.

you can also make edges with differnt brush textures everywhere. that's how marco bucci does edges.

Use poly lasso tool to make shapes.

1. start randomly losing edges
2. start with an area of already low contrast
3. start softening and losing some edges
4. testing how soft or lost i can go, purposely push it too far, and then bring back some edges
5. if an area gets too soft or lost, you can bring back hard edges where you want them
6. balance it so some areas are more visually important, some are less
