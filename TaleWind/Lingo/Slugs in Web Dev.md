example.co m/blog/==**what-is-a-slug**==

A substitute for numerical database IDs.
Lowercase letters, number, and hyphens.

Generated automatically from titles in CMS platforms like Wordpress.

Synonyms: URL path, URL identifier, human-readable URL, permalink suffix

## Wordpress

**Permalink (Path & Slug):** The portion after the domain that identifies the specific page or post.

- **Path:** The directory structure (e.g., `/blog/` or `/category/`).
- **Slug:** The unique, user-friendly string for a specific post (e.g., `wordpress-url-anatomy`).
